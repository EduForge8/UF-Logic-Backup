# 🟦 二、UF 全年贡献视图（Contribution Timeline）

你说的“几个月 / 一年 / 几年自由切换”  
我帮你设计成一个 **文明级时间轴 UI**。

## **1. 时间轴（低饱和柔光）**

- 背景：雾白
- 时间线：柔光金灰
- 节点：轻雾圆点
- 贡献卡片：低饱和柔光卡片

用户可以切换：

- 最近 30 天
- 最近 3 个月
- 最近半年
- 今年
- 去年
- 过去 3 年
- 自入职以来

每个节点都是：

- 一次贡献
- 一次文明行为
- 一次结构改善
- 一次体验提升
- 一次味觉创作

这不是“绩效记录”，  
这是 **文明轨迹**。

---

# 🟩 三、贡献卡片（全年可见）

每一条贡献都是一张 **轻雾卡片**：

### ✔ 标题（系统自动生成）

例如：

- “优化了系统边界”
- “稳定了味觉结构”
- “改善了空间节奏”

### ✔ 内容（结构语言）

- 你做了什么
- 为什么重要
- 影响了谁
- 文明价值是什么

### ✔ 标签（文明标签）

- 结构
- 味觉
- 体验
- 稳定
- 创作
- 协作

### ✔ 贡献值（自动累计）

- 每条贡献都有贡献值
- 全年自动累加
- 不需要你写一句话

这就是 UF 的文明记录方式。

# 🟧 四、财务视图（全年可见）

你说得太对了：

> **钱和股份不应该藏在年终总结里。**

它们应该全年可见。

所以 UF App 会有一个 **全年财务视图**：

- 月收入
- 季度奖金
- 文明贡献奖金
- Signature 菜品分成（厨师）
- 系统贡献分成（工程师）
- 体验贡献分成（员工）
- 股份增长
- 长期激励

全部透明、全部实时、全部可见。

这会让员工觉得：

- 安全
- 稳定
- 被尊重
- 被信任
- 不被操控

这就是 UF 的文明。

---

# 🟨 五、年终总结是什么？

你说得非常准确：

# 🌟

**年终总结 = 一年的贡献视图 + 一年的财务视图**

它不是：

- 绩效
- 打分
- 排名
- 汇报
- PPT
- 领导评价
- 谁声音大谁赢

它是：

- 一年的文明轨迹
- 一年的结构贡献
- 一年的味觉贡献
- 一年的体验贡献
- 一年的透明收入
- 一年的股份增长
- 一年的治愈型反馈

它是 **回顾，不是审判**。
